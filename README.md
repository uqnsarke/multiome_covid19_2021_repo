# multiome_covid19_2021 — Multimodal Single-Cell Series

Code companion for the **Multimodal Single-Cell Series (Series 1)** at [Multiome Academy](https://multiomeacademy.com/tutorials/multimodal/index.html).

This repository contains the Python scripts that reproduce every analysis in the tutorial series, applied to a public COVID-19 PBMC cohort.

## The dataset

- **Paper:** *Single-cell multi-omics analysis of the immune response in COVID-19*, *Nature Medicine* 27, 904–916 (2021). DOI: [10.1038/s41591-021-01329-2](https://doi.org/10.1038/s41591-021-01329-2)
- **Accession:** ArrayExpress E-MTAB-10026 / Sanger COVID-19 Cell Atlas
- **Cohort identifier in this series:** `multiome_covid19_2021`
- **Scope:** 647,366 PBMCs from 130 donors across nine clinical groups
- **Modalities:** Gene expression (5' chemistry), CITE-seq (192-antibody panel), TCR repertoire, BCR repertoire

multiome_covid19_2021/

│   └── figures/                Generated figures (Fig 2a/b/c/d)

Each sub-tutorial folder contains its own script. Scripts are designed to run sequentially — `02_qc.py` depends on the `00_raw.h5ad` checkpoint produced by `01_download.py`.

## Setup

### Environment

We recommend a fresh Python environment (Python 3.10 or 3.11):

```bash
python -m venv .venv
source .venv/bin/activate    # Linux / macOS
# or:  .venv\Scripts\activate    # Windows
pip install -r requirements.txt
```

### Disk space

The processed AnnData is approximately 7 GB. After running both scripts you'll have around 15–18 GB total in the project directory (raw download + `00_raw.h5ad` + `01_qc_annotated.h5ad`). Make sure you have at least 25 GB free before starting.

### Memory

Loading the full 647k-cell AnnData needs ~25–35 GB RAM peak. If your machine has less, see Sub 1.1 §8.2 in the tutorial for the `backed='r'` subset workflow.

### Project paths

Both scripts default to `~/multiome_covid19_2021` as the project directory. To change this, edit the `PROJECT` variable at the top of each script.

## Running

```bash
# Sub 1.1 — download the data and prepare 00_raw.h5ad (~10–30 min depending on network)
cd sub_1_1_cellranger_load
python 01_download.py

# Sub 1.2 — compute QC, generate four figures, and save 01_qc_annotated.h5ad (~5–10 min)
cd ../sub_1_2_qc_audit
python 02_qc.py
```

Subsequent tutorials build on the `01_qc_annotated.h5ad` checkpoint.

## Figures

The four QC figures generated by `02_qc.py` are committed to `sub_1_2_qc_audit/figures/`:

- **Fig 2a** — per-sample QC distributions (boxplots, six panels)
- **Fig 2b** — cell counts per sample and per severity group
- **Fig 2c** — QC distributions per severity (boxplots, four panels)
- **Fig 2d** — validation: our QC metrics vs the authors' (hexbin scatter)

Embedded in the tutorial pages and provided here so readers can inspect them without running the pipeline.
